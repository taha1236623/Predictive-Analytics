# Predictive Analytics Using Historical Data

## Objective
Predict future sales using historical retail transaction data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps
1. Loaded the Excel dataset.
2. Cleaned the data.
3. Created the TotalPrice column.
4. Trained a Linear Regression model.
5. Predicted sales.
6. Displayed actual vs predicted sales in a graph.

## Output
The graph displays actual sales (blue dots) and predicted sales (red line).
